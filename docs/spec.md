# CTF Diary — 功能规格说明书

> 版本: v1.0 | 日期: 2026-06-02 | 状态: 草稿

---

## 1. 项目概述

**CTF Diary** 是一个桌面端 CTF 选手日常工具，集比赛管理、附件管理、日志打卡、笔记编辑于一体。界面美观，支持自定义背景图片。

## 2. 用户故事

### 2.1 比赛管理

**作为** CTF 选手
**我想要** 从 CTFtime 获取近期比赛信息
**以便** 规划我的参赛安排

验收标准:
- 展示 CTFtime 上近期比赛列表（名称、日期、权重、格式、链接）
- 提供"我要参加"按钮，点击后自动为比赛创建本地目录
- 区分"已参加"和"未参加"的比赛
- 支持手动添加/删除比赛

### 2.2 附件管理

**作为** CTF 选手
**我想要** 为每场比赛管理 PWN 和 RE 的附件文件
**以便** 整理和快速定位解题材料

验收标准:
- 每场比赛目录下自动创建 `pwn/` 和 `re/` 子目录
- 支持拖拽/导入附件文件
- 支持文件浏览和搜索
- 支持分类标签（binary, script, notes 等）

### 2.3 日志与打卡

**作为** CTF 选手
**我想要** 记录每天的学习内容和比赛进展
**以便** 追踪我的成长轨迹

验收标准:
- 日历视图，可视化打卡记录
- 每天可以写学习日志（支持 Markdown）
- 提供打卡（check-in）功能
- 统计连续打卡天数、总打卡数
- 日志支持标签分类（如: pwn, reverse, web, crypto, misc）

### 2.4 Markdown 编辑与解析

**作为** CTF 选手
**我想要** 使用 Markdown 编写和阅读解题笔记
**以便** 写出结构清晰的 writeup

验收标准:
- 内置 Markdown 编辑器（实时预览）
- 支持代码高亮（至少 Python/C/汇编/bash）
- 支持图片插入
- 支持文件链接引用

### 2.5 Python 解析

**作为** CTF 选手
**我想要** 在工具内运行/测试 Python exploit 脚本
**以便** 快速验证思路

验收标准:
- 内嵌 Python 代码查看器（语法高亮）
- 支持一键运行 Python 脚本（沙箱化）
- 输出结果在工具内展示
- 支持管理多个 Python 脚本版本

### 2.6 个性化设置

**作为** 用户
**我想要** 自定义工具的外观
**以便** 让工具更符合我的审美

验收标准:
- 自定义背景图片（支持本地图片上传）
- 多套预设主题/壁纸
- 字体大小/行高调整
- 界面语言切换（中/英）
- 设置持久化保存

## 4. 技术约束

- 桌面应用，支持 Windows 平台
- 本地优先，数据存储于本地文件系统
- 界面响应式、现代化
- 需访问 CTFtime API (https://ctftime.org/api/v1/)

## 5. 非功能需求

- 启动速度 < 3 秒
- 界面流畅无卡顿
- 数据本地存储，无隐私泄露
- 离线可用（比赛信息支持缓存）
